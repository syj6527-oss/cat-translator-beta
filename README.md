# 🐱 Translator

SillyTavern용 맞춤형 번역 확장 프로그램

Gemini API 기반의 스마트 번역기로, RP(롤플레이) 환경에 최적화된 번역을 제공합니다.

---

## ✨ 주요 기능

### 번역
- **수동/자동 번역** — 🐱 버튼 클릭 또는 자동 번역 모드 (입력/출력/둘 다)
- **대사 병기 번역** — 대사는 원문 유지 + 한국어 번역 병기, 지문은 한국어 통번역
- **양방향 번역** — 한↔영 / 한↔일 / 한↔중 자동 감지
- **벌크 번역** — 최근 N개 메시지 일괄 번역 (⚡ 버튼)
- **재번역** — 같은 메시지를 다른 표현으로 재번역, 번역 히스토리에서 선택 가능
- **원문 복구** — 되돌리기(↩) 버튼으로 원문 즉시 복구

### 대사 병기 모드
```
원문: He clenched his jaw. "I bought a mattress for you."
병기: 그는 이를 악물었다. "I bought a mattress for you. [널 위해 매트리스를 샀어.]"
```
- `""` 안의 대사 → 원문 유지 + `[한국어 번역]` 병기
- `""` 밖의 지문 → 한국어 통번역
- 한영 / 한일 / 한중 병기 지원

### 스마트 기능
- **IndexedDB 캐시** — 동일 문장 재번역 시 API 호출 없이 캐시 히트 (30일 보존)
- **사전 (Glossary)** — 고유명사/용어 사전 등록 (드래그로 즉시 등록 가능 🐾)
- **문맥 번역** — 이전 메시지를 참조하여 문맥에 맞는 번역
- **번역 스타일** — 일반 / 소설 / 캐주얼 프리셋
- **정규식 인포박스 보존** — 정규식 확장의 트리거 패턴을 번역 시에도 유지
- **AI 생성모드 감지** — 번역 대신 RP 이어쓰기를 하는 비정상 출력 자동 차단

### 테마
- 🐱 **고양이 모드** (Flash 모델) — 주황 글로우, 🐟 완료 이모지
- 🐯 **호랑이 모드** (Pro 모델) — 금색 글로우, 🍖 완료 이모지
- 모델 전환 시 자동 테마 변경

## ⚙️ 설정 방법

### 기본 권장 설정

| 설정 항목 | 값 |
|---|---|
| 자동 번역 | 꺼짐 (수동 추천) |
| 양방향 번역 | 꺼짐 |
| 대사 병기 | 한영 병기 |
| 목표 언어 | 아무거나 (병기 ON 시 자동 결정) |

### 대사 병기 사용 시
- **입력**: 한국어 작성 → 🐱 클릭 → 순수 영어로 변환 → AI에게 전송
- **출력**: AI 영어 응답 → 🐱 클릭 (또는 자동) → 대사 병기 번역 적용
- 대사 병기 꺼짐 시 기존 통번역과 100% 동일하게 작동

### 번역 품질 개선 팁
- **추가 지시사항** 칸에 말투/어조 지정 가능
  - 예: `항상 반말로 번역하세요. 예시) You are beautiful. → 너 예쁘다.`
- **사전** 등록으로 고유명사 일관성 확보
  - 예: `Ghost=고스트`, `Soap=소프`
- **문맥 범위** 조절로 번역 정확도 향상 (기본 1, 최대 6)

## 🔧 연결 방법

### 방법 1: 연결 프로필 (권장)
SillyTavern의 연결 관리자에서 Gemini 프로필을 설정한 후, 드롭다운에서 선택

### 방법 2: 직접 연결
1. [Google AI Studio](https://aistudio.google.com/apikey)에서 API Key 발급
2. 설정창 → API Key 칸에 입력
3. 모델 선택 (Flash 또는 Pro)

### 지원 모델
| 모델 | 특징 |
|---|---|
| Gemini 2.0 Flash | 빠른 속도, 일반 번역 |
| Gemini 2.5 Flash | 빠른 속도 + 병기 안정적 |
| Gemini 2.0 Pro | 높은 품질, 복잡한 포맷 |
| Gemini 3.1 Pro | 최고 품질, 긴 문장 병기 안정적 |
| Vertex AI | Google Cloud 연동 |

## 📋 대사 병기 옵션

| 옵션 | 입력 방향 | 출력 방향 |
|---|---|---|
| 꺼짐 | 목표 언어 따름 | 목표 언어 따름 |
| 한영 병기 | 한국어 → English | English 대사 유지 + [한국어] |
| 한일 병기 | 한국어 → Japanese | Japanese 대사 유지 + [한국어] |
| 한중 병기 | 한국어 → Chinese | Chinese 대사 유지 + [한국어] |

## ⚠️ 참고 사항

- 대사 병기는 프롬프트 기반이므로 간헐적으로 `[]` 누락 가능 → 재번역으로 해결
- 캐릭터 프로필/시스템 프롬프트는 **영어로 유지** 권장 (한국어 지시 시 AI 출력 언어 혼동 가능)
- 정규식 확장의 인포박스 트리거 패턴은 번역 시 보존됨
- 병기 모드 전환 시 캐시 키가 자동 분리되어 기존 캐시와 충돌 없음
