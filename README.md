# 🐱 Translator

**SillyTavern용 스마트 번역 확장 프로그램**

Gemini API / Vertex AI를 활용한 롤플레이 특화 번역기입니다.
자동 언어 감지, 사전 시스템, 모델별 캐시 분리 등 롤플 환경에 최적화된 기능을 제공합니다.

---

## ✨ 주요 기능

### 번역
- **스마트 언어 감지** — 한국어 ↔ 영어 자동 판별, 일본어·중국어도 지원
- **경량/풀 프롬프트 자동 전환** — 짧은 문장은 가볍게, 긴 문장은 정밀하게
- **HTML/마크다운/YAML 구조 보존** — 태그, 서식, 들여쓰기 그대로 유지
- **재번역** — 같은 문장을 다른 표현으로 다시 번역 가능
- **번역 히스토리** — 3회 이상 번역된 문장은 히스토리 팝업에서 선택 가능

### 전체(벌크) 번역
- 📋 전체 / 🦁 20개 / 🐱 15개 / 🐱 10개 / 🐭 5개 / ✏️ 직접 입력
- 이미 번역된 메시지는 자동 스킵
- 진행률 토스트 표시 + 클릭으로 중단 가능

### 사전 시스템
- **원문=번역어** 형태로 등록 (예: `Ghost=고스트`)
- **드래그 등록** — 채팅에서 텍스트 드래그 → 🐾 아이콘 → 바로 등록
- **같은 단어 다른 뜻** 등록 가능 (동일 쌍만 중복 차단)
- AI가 문맥에 맞게 형태소 변화를 자동 적용

### 캐시
- **IndexedDB 영구 캐시** — 브라우저 종료 후에도 유지
- **모델별 캐시 분리** — Flash와 Pro 번역 결과가 섞이지 않음
- **즐겨찾기 핀** — 핀된 번역은 캐시 만료에서 제외 (30일)
- **캐시 히트율/절약 토큰 통계** 실시간 표시

### 🐱 고양이 / 🐯 호랑이 모드
- Flash 계열 → 🐱 고양이 모드 (오렌지 글로우)
- Pro 계열 → 🐯 호랑이 모드 (골드 글로우)
- 모델 변경 시 자동 테마 전환 + 알림

## 📦 설치

### SillyTavern 확장으로 설치

1. SillyTavern 실행
2. 확장 관리자 → **URL로 설치**
3. 아래 URL 입력:

```
https://github.com/YOUR_USERNAME/cat-translator
```

4. 설치 완료 후 SillyTavern 새로고침

### 수동 설치

```bash
cd SillyTavern/data/default-user/extensions/
git clone https://github.com/YOUR_USERNAME/cat-translator
```

SillyTavern 재시작 후 확장 패널에서 🐱 Translator 확인.

## ⚙️ 설정

### API 연결 (택 1)

| 방식 | 설명 |
|------|------|
| **연결 프로필** | SillyTavern Connection Manager 프로필 선택 |
| **Gemini 직접 연결** | Google AI Studio API Key 입력 |
| **Vertex AI** | Vertex API Key 입력 (global 기본, 실패 시 프로젝트 ID/리전 입력) |

### 모델 선택

| 그룹 | 모델 | 특징 |
|------|------|------|
| 🐱 Flash | 1.5 Flash, 2.0 Flash | 빠름, 일반 번역에 충분 |
| 🐯 Pro | 1.5 Pro, 2.0 Pro Exp | 뉘앙스 우수, 느림 |
| 🐱 Vertex Flash | Vertex 2.0/1.5 Flash | Vertex 인프라 경유 |
| 🐯 Vertex Pro | Vertex 2.0/1.5 Pro | Vertex 인프라 경유 |
| ✏️ 직접 입력 | 커스텀 모델명 | 새 모델 출시 시 활용 |

### 주요 설정값

| 항목 | 권장값 | 설명 |
|------|--------|------|
| 온도 | 0.3 | 낮을수록 정확, 높을수록 창의적 (0.0~1.0) |
| 토큰 | 8192 | 최대 출력 토큰 (최대 20000) |
| 문맥 범위 | 1~3 | 이전 메시지 참조 수 (Flash: 3 이하, Pro: 6까지) |
| 스타일 | 일반 번역 | 일반/소설/캐주얼 |

> 모든 설정은 변경 시 자동 저장됩니다.

## 🎮 사용법

### 메시지 번역
- 캐릭터 이름 옆 🐱/🐯 클릭 → 번역
- ↩️ 클릭 → 원문 복구
- 이미 번역된 메시지 클릭 → 히스토리에서 선택 또는 새로 번역

### 입력창 번역
- 입력창 옆 🐱/🐯 → 입력 중인 텍스트 번역
- ↩️ → 원문 복구
- ⚡ → 전체(벌크) 번역 팝업

### 사전 드래그 등록
1. 채팅 영역에서 텍스트 드래그
2. 🐾 아이콘 클릭
3. 번역어 입력 → 등록
